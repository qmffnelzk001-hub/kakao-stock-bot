# 🚀 카카오톡 주식 봇 사용 설명서 (Walkthrough)

이 프로젝트는 카카오톡 채널을 통해 주식 정보를 조회하고, AI가 뉴스를 분석해주는 서비스입니다.

## 🛠️ 설치 및 실행 방법

1.  **서버 실행**:
    터미널에서 아래 명령어를 입력하여 서버를 실행합니다.
    ```bash
    node index.js
    ```

2.  **외부 노출 (Tunneling)**:
    카카오톡 서버가 로컬 서버에 접속할 수 있도록 `ngrok` 등을 사용하여 주소를 생성해야 합니다.
    ```bash
    ngrok http 3000
    ```
    생성된 주소(예: `https://abcd-123.ngrok-free.app/stock`)를 복사하세요.

3.  **카카오 i 오픈빌더 설정**:
    - [카카오 i 오픈빌더](https://chatbot.kakao.com/)에 접속하여 시나리오를 생성합니다.
    - **스킬(Skill)** 탭에서 새로운 스킬을 생성하고, URL에 위에서 복사한 주소를 입력합니다.
    - **시나리오**에서 사용자의 발화(예: `주식 : 삼성전자`)를 설정하고, 응답으로 방금 만든 스킬을 선택합니다.

## 💬 사용 예시

카카오톡 채팅창에 아래와 같이 입력해 보세요!

> **주식 : 삼성전자**

### 🤖 봇의 응답:
1.  **현재 주가**: 삼성전자의 현재 가격과 등락률을 보여줍니다.
2.  **호재 뉴스**: AI가 분석한 최근의 긍정적인 뉴스 요약.
3.  **악재 뉴스**: AI가 분석한 최근의 부정적인 뉴스 요약.
4.  **뉴스 링크**: 직접 확인할 수 있는 뉴스 기사 링크들.

## ⚠️ 주의 사항
- 현재는 별도의 네이버 API 키 없이 구글 뉴스 RSS를 활용하고 있습니다. 더 정확한 검색 결과를 원하시면 네이버 검색 API를 연동하는 것을 권장합니다.
- `yahoo-finance2`는 해외 및 국내 주식을 지원하지만, 국내 주식의 경우 종목명 검색이 가끔 미흡할 수 있으니 `주식 : 005930.KS`와 같이 티커를 직접 입력할 수도 있습니다.
