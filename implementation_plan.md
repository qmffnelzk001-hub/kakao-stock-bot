# 주식 알림 카카오톡 봇 개발 계획 (Implementation Plan)

카카오톡 채널을 통해 주가 정보 및 호재/악재 뉴스를 제공하는 봇을 구축합니다.

## 1. 개요
사용자가 `주식 : 삼성전자`라고 입력하면 다음과 같은 정보를 제공합니다.
- 현재 주가 및 등락 (Yahoo Finance API 활용)
- 최신 뉴스 수집 (네이버 또는 Google 뉴스)
- 뉴스 분석 (Gemini AI 활용: 호재/악재 분류 및 요약)
- 뉴스 링크 제공

## 2. 기술 스택
- **Runtime**: Node.js
- **Server**: Express.js (카카오 I Open Builder 웨이백 연동)
- **AI**: Google Gemini API (뉴스 분석)
- **Data**: 
    - `yahoo-finance2`: 주식 시세 데이터
    - `axios`: 뉴스 API 또는 RSS 수집
- **Framework**: 카카오 i 오픈빌더 (카카오톡 채널 챗봇 전용)

## 3. 구현 단계
1. **서버 초기화**: `kakao-stock-bot` 디렉토리 신설 및 기본 프로젝트 설정
2. **카카오 챗봇 프로토콜 구현**: 스킬(Skill) 서버 규격에 맞는 응답 구조 생성
3. **주식 데이터 연동**: 종목명으로 종목 코드(`Ticker`)를 찾고 주가를 가져오는 로직 구현
4. **뉴스 분석 시스템**: 
    - 검색 API를 통해 뉴스 수집
    - Gemini API를 활용하여 감성 분석(호재/악재) 및 요약 로직 구현
5. **테스트 및 검증**: 로컬 서버 시뮬레이션 및 응답 확인

## 4. 필요한 설정 (사용자 준비 사항)
- **카카오 i 오픈빌더** 아이디 및 채널 생성
- **Gemini API Key** (기존 것 활용 가능)
- **네이버 검색 API Key** (뉴스 검색용, 권장)
